Part I: What is Bootstrap?

1. Bootstrap is a framework. A framework, or software framework, is a platform for developing software applicationsLinks to an external site.. It provides a foundation on which software developers can build programs for a specific platformLinks to an external site.. For example, a framework may include predefined classesLinks to an external site. and functionsLinks to an external site. that can be used to process inputLinks to an external site., manage hardware devices, and interact with system softwareLinks to an external site.. This streamlines the development process since programmers don't need to reinvent the wheel each time they develop a new application.
2. What You Can Do with Bootstrap? There are lots of things you can do with Bootstrap. You can easily create responsive websites. You can quickly create multi-column layout with pre-defined classes. You can quickly create different types of form layouts. You can quickly create different variation of navigation bar. You can easily create components like accordions, modals, etc. without writing any JS code. You can easily create dynamic tabs to manage large amount of content. You can easily create tooltips and popovers to show hint text. You can easily create carousel or image slider to showcase your content. You can quickly create different types of alert boxes. The list does not end here, there are many other interesting things that you can do with Bootstrap. 3. At this point you have been using bootstrap 5, but probably haven't really been aware of what is actually doing. One big advantage to using bootstrap is how you can make your web page be responsive, meaning you can view the web site on many different devices(different sized screens) and you will have the same user experience.

Part II: Setup your webpage to use Bootstrap

4. In order to use any of the classes contained within Bootstrap we need reference the Bootstrap files. Bootstrap will be treated as External styling. Just like we learned with Version 3.0 in order to use any of the styling information from an external file we need to reference these files. We have 2 choices; #1 we can download the file and host them in our own CSS folders, #2 We can reference them from a hosted website. Typically you will find that most Bootstrap code is referenced using a hosted website. 5. This line of code(below) is the main file that Bootstrap will be using. For now we will just get used to loading all our resources that we will need in the future. <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.2/css/bootstrap.css">
